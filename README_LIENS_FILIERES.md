# Paramétrage des liens de filières

L'onglet `liens_filieres` du fichier `parametres_simulateur_inphb.xlsx` contient :

- `filiere` : code utilisé par le moteur ;
- `ecole` : école ou structure de rattachement ;
- `cycle` : cycle court ou préparatoire ;
- `intitule` : libellé développé affiché dans les cartes ;
- `url_officielle` : page ou plaquette officielle ouverte au clic ;
- `type_lien` et `commentaire` : informations de maintenance.

Pour mettre à jour une URL, modifie uniquement la cellule correspondante dans l'Excel. Aucun changement Python n'est nécessaire.
